# 🩺 Doctor Appointment Website

A modern and responsive web application for a **General Practitioner Clinic**, built using React and TailwindCSS.
This project showcases a complete medical website including doctor presentation, services, appointment booking, and contact features.

---

## 📌 Features

* 🏥 Doctor profile presentation
* 📋 Services overview
* 📅 Appointment booking page
* 📝 Blog section
* 📞 Contact page (WhatsApp, Email, etc.)
* 💻 Fully responsive design
* ⚡ Clean UI inspired by modern medical websites

---

## 🛠️ Technologies Used

* ⚛️ React (with TypeScript)
* 🎨 Tailwind CSS
* 📦 Vite
* 📁 Component-based architecture

---

## 📂 Project Structure

```
src/
│
├── components/   # Reusable UI components
├── hooks/        # Custom React hooks
├── lib/          # Utility functions
├── pages/        # Main pages (Home, About, Services, etc.)
│
├── App.tsx
├── main.tsx
```
